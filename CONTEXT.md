# Sound Selection

Macのシステム全体で使う音声入力デバイスと音声出力デバイスを選ぶ文脈。

## Language

**音声入力デバイス**:
Macが既定の入力先として使うマイク、入力機器、または仮想・集約された入力デバイス。
_Avoid_: 入力ソース

**音声出力デバイス**:
Macが既定の通常音声の出力先として使うスピーカー、ヘッドホン、出力機器、または仮想・集約された出力デバイス。システム効果音や通知音の出力先とは区別する。
_Avoid_: 出力ソース

**選択可能な音声デバイス**:
現在Macに接続されていて、対象の方向で実際に既定デバイスとして選べる音声デバイス。入力一覧には入力できるデバイスだけ、出力一覧には通常音声を出力できるデバイスだけを含める。
_Avoid_: 認識済みデバイス, 過去のデバイス

**現在の音声デバイス**:
実行時点でMacの既定として設定されている音声入力デバイスまたは音声出力デバイス。
_Avoid_: 最後に選んだデバイス

**確定**:
選んだ音声入力デバイスと音声出力デバイスをMacの設定へ反映する意思表示。番号で選ぶだけでは確定ではなく、明示的なOKだけを確定として扱う。
_Avoid_: 即時反映

**キャンセル**:
選択途中または確認画面で、選んだ音声デバイスをMacの設定へ反映せずに終了する意思表示。
_Avoid_: 失敗

**部分反映**:
音声入力デバイスまたは音声出力デバイスの片方だけがMacの設定に反映された状態。このツールでは正常な確定結果とは扱わない。
_Avoid_: 部分成功

## Example dialogue

Developer: 「入力ソースを一覧に出しますか？」

Domain expert: 「この文脈では入力ソースではなく、音声入力デバイスと呼びます。現在選択可能な音声入力デバイスだけを一覧に出します。」

Developer: 「出力は通知音も同じデバイスに変えますか？」

Domain expert: 「いいえ。音声出力デバイスは通常音声の出力先だけを指します。システム効果音や通知音の出力先とは区別します。」

Developer: 「番号を選んだらすぐ反映しますか？」

Domain expert: 「いいえ。番号選択はまだ確定ではありません。明示的なOKだけが確定で、キャンセルなら何も反映しません。」
